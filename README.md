# Gear-WenXinWorkShop-Starter

百度"文心千帆 WENXINWORKSHOP"模型的Starter，帮助您可以快速使用百度的AI能力，一行代码即可调用百度文心千帆大模型。
支持的模型已有ErnieBot、ErnieBot-Turbo模型，下版本将添加更多模型支持，目前已完整支持“文心一言”大模型的API。
